# rftinternship_day12

This project is a simple Student Performance Dashboard created using Python, Matplotlib, and NumPy.
The program visualizes students' marks using:
Simple Bar Chart
Grouped Bar Chart
It helps in comparing student performance across multiple subjects.

Concepts Used
1. Matplotlib
Used for creating charts and graphs.
2. NumPy
Used for numerical operations and bar positioning.
3. Bar Chart
Displays marks of students visually.
4. Grouped Bar Chart
Compares marks of multiple subjects side by side.
5. Data Visualization
Represents data graphically for better understanding.


Features
 Simple Bar Chart
 Grouped Bar Chart
 Multiple Subject Comparison
 Data Visualization
✅ Student Performance Analysis
